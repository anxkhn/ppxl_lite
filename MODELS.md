# Perplexity AI Models Reference

Complete list of available models organized by search mode.

## Subscription Tiers

| Tier | Price | Description |
|------|-------|-------------|
| Free | $0 | 5 Pro searches per 4 hours, no Deep Research or Labs |
| Pro | $20/month | 300+ Pro searches/day, limited Deep Research and Labs |
| Max | $40/month | Unlimited everything, access to frontier models |

## Search Modes

| Mode | Description | Limits |
|------|-------------|--------|
| auto | Basic search using Turbo model | Unlimited |
| pro | Advanced search with model selection | Unlimited for Pro/Max |
| reasoning | Complex reasoning with thinking models | Unlimited for Pro/Max |
| deep research | Extended research for in-depth analysis | Limited for Pro, unlimited for Max |
| labs | Multi-step tasks with troubleshooting | 50/month for Pro, unlimited for Max |
| study | Fast routine research (new) | No limit mentioned |

## Model Naming Convention

Backend values use:
- Lowercase letters
- No spaces or hyphens
- Numbers directly appended
- Underscores for multi-word names

Examples:
- `gpt52` not `gpt-5.2`
- `claude45sonnet` not `claude-4.5-sonnet`
- `o3pro_research` not `o3-pro-research`
